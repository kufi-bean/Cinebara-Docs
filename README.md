# Welcome

These are the docs & plans for Cinebara.

# What is "Cinebara"?

It's an app intended for making films in VR. It intends to offer a more accomodating experience for all fields of film making, but is still in very early development. You can look around to see what we have, but many things lack implementation right now. If you are curious what Cinebara intends to have once it is mature, consider browsing the [Features](features.md) page.
